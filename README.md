# Rynnet

A modern, fast web proxy built with Scramjet.

## Features

- Clean, minimal interface
- Dark/Light theme support
- Multiple search engine options
- Encrypted traffic routing
- Fast performance

## Quick Start

Install dependencies:
```bash
npm install
```

Start the server:
```bash
npm start
```

Open `http://localhost:8080` in your browser.

## Configuration

Settings are accessible via the settings button in the top right:
- Choose your preferred search engine
- Toggle between dark and light themes

## Tech Stack

- Scramjet proxy engine
- Fastify web server
- Wisp WebSocket protocol
- Libcurl transport layer
- Bare-mux for transport management
